void Loot.Modifiers.ModifierCachePlayer.UpdateAttachments ( )
private

This method handles updating all delegations First, it will perform 'manual' detachments calling ModifierEffect.DetachDelegations(ModifierPlayer) then it will perform 'manual' attachments calling ModifierEffect.AttachDelegations(ModifierPlayer) These will unbind/bind ModifierEffect.ResetEffects to ModifierPlayer.OnResetEffects and set ModifierEffect.IsBeingDelegated to false or true.

After this, the automatic detachments/attachments are done which will bind any methods to whichever method noted in their AutoDelegation attribute.
